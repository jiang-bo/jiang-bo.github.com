1.确保本机上已经装了Xcode，可以从随机附带的DVD中获取，或者到 Apple Developer Connection下载安装。2.从 shell script from topfunky.net (mirror) 下载安装脚本，该脚本会自动完成memcached的安装，包括相应的依赖包
3.执行下载的脚本， 

$ chmod +x install-memcached.sh
$ sudo ./install-memchaced.sh
4.在环境变量里添加EVENT_NOKQUEUE=1
5.验证是否成功， $memcached -h，应该产生帮助提示 6.启动memcached, 
$memcached -m 32 -p 11211 -d
